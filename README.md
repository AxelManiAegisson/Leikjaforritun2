# Leikjaforritun2
Leikjaforritun í Tækniskólanum

Hugmynd 1.

Setting:
Leikur gerist í geiminum.

Gameplay:
Maður stjórnar geimskipi og skítur óvini. Lendir á plánetu til að fá eldsneyti og/eða súrefni eða eitthver önnur resources.
